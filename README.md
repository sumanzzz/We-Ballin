# Save The Ball

A simple 2D survival game built with **Lua** and **LÖVE (Love2D)**.

## Overview

Control a ball using the mouse cursor and survive as long as possible. Enemy balls spawn as you progress through the game and actively chase the player. As the game continues, more enemies appear, increasing the difficulty and making survival more challenging.

## Features

* Mouse-controlled movement
* Enemy balls that chase the player
* Progressive difficulty scaling
* Collision detection
* Survival-based gameplay

## Controls

| Action    | Control      |
| --------- | ------------ |
| Move Ball | Mouse Cursor |

## Technologies Used

* Lua
* LÖVE (Love2D)

## Objective

Avoid colliding with enemy balls and survive for as long as possible.
